# E-Commerce Tomato

A modern and responsive food delivery platform that allows users to browse, search, and order a variety of food items online. The website is designed to provide a seamless user experience with intuitive navigation and appealing visuals.

---

## Features
- **Dynamic Layout**: Clean, responsive design that adapts to all devices.
- **Product Listings**: Categorized display of food items with detailed descriptions and images.
- **Search Functionality**: Easily search for specific food items.
- **Cart System**: Add items to the cart and proceed to checkout.
- **Smooth Navigation**: Intuitive navigation bar for quick access to all sections.

---

## Technologies Used
- **HTML**: Structured the content and layout of the website.
- **CSS**: Styled the elements for a visually appealing interface.
- **ReactJS**: Implemented dynamic features such as the search and cart system.
- **Netlify**: Deployed the website for fast and reliable hosting.

---

## How to Access
The website is live at: [E-Commerce Tomato](https://ecommercetomato.netlify.app/)
